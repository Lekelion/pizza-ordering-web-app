# Pizza Ordering Web App

A Java Spring Boot web application for managing pizza orders, customer login, and order processing.

## Features
- Customer login flow
- Pizza ordering interface
- Form validation
- Order persistence with JPA
- MVC architecture with Thymeleaf views
- Database-backed customer and order management

## Tech Stack
- Java 17
- Spring Boot
- Spring MVC
- Spring Data JPA
- Thymeleaf
- MySQL

## How to Run
1. Clone the repository
2. Open the project in IntelliJ IDEA or your preferred Java IDE
3. Configure the MySQL database connection
4. Run the SQL setup scripts if included
5. Start the Spring Boot application
6. Open the application in your browser
